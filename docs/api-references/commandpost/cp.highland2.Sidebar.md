# cp.highland2.Sidebar

Represents the sidebar for a document opened in Highland.

## API Overview
* Fields - Variables which can only be accessed from an object returned by a constructor
 * [document](#document)
 * [doHide](#doHide)
 * [doShow](#doShow)
